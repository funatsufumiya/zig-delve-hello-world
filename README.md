# zig-delve-hello-world

Tiny example of using [Delve Framework](https://github.com/Interrupt/delve-framework) as an external library.

Tested with Zig 0.13.0.

## Run

```bash
# debug:
$ zig build run

# release:
$ zig build run -Doptimize=ReleaseSafe
```
